# License Decision

The repository is intentionally public, but no final license is granted yet. Public visibility allows people to read and fork the repository through GitHub, but it does not automatically grant broad rights to use, redistribute, sell, or relicense the code and assets.

The owner’s current intent is **source-available for noncommercial use**: people may eventually be allowed to study, copy, and modify the project, but not sell it, monetize derivatives, or use it commercially without separate permission. That restriction means the project would not be OSI open source.

Before the first external playtest release, choose and apply a clear license strategy, likely one of:

1. A recognized noncommercial source-available software license for code, with separate content/art terms.
2. A custom license reviewed by counsel.
3. All rights reserved until collaboration or distribution needs justify a broader grant.

Until that decision is completed, all rights are reserved except those required by GitHub’s Terms of Service to host and display the repository.
