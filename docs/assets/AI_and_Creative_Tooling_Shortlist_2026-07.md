# AI and Creative Tooling Shortlist — July 2026

**Version:** 1.0
**Status:** Active procurement and workflow guidance
**Review date:** July 24, 2026

## 1. Current available subscriptions

### ChatGPT Plus

Use for:

- project planning and release management;
- GitHub-native documentation and code;
- prompt and production-brief authoring;
- image generation and image editing;
- file generation and analysis;
- review of external media outputs;
- provenance and asset-register maintenance.

### Google AI Plus

The project owner's Gemini usage screen identifies the account as `PLUS`.

Google's current plan documentation describes AI Plus as providing:

- 2x standard Gemini usage limits;
- a 128K-token context window;
- access to Gemini media features according to current availability;
- Google Flow and Flow Music access;
- compute-based limits that refresh during the day until the weekly limit is reached.

The account currently shows separate current-period and weekly usage meters. An AI Pro promotional upgrade is visible, but the project does not currently require the higher tier.

Recommended uses:

- visual concept exploration;
- environment and prop variations;
- storyboard and short-motion experiments in Flow;
- temporary music sketches in Flow Music;
- independent review of large design documents;
- alternative prompt interpretation for difficult assets.

Do not upgrade to AI Pro until a measured Plus limitation blocks an approved work package.

## 2. Voice, narration, and sound

### ElevenLabs

Current public tiers include:

- Free;
- Starter at approximately $6 per month;
- Creator at approximately $22 per month, with a first-month discount sometimes offered;
- higher professional tiers.

Recommended project use:

- Underteller voice-design auditions;
- temporary narration;
- short sound effects and loops;
- pronunciation and pacing comparison;
- voice consistency experiments.

Purchase recommendation:

- stay on Free during script preparation;
- buy one Starter month for the first bounded audition and SFX package;
- consider Creator only if the approved line count exceeds Starter capacity.

Do not imitate or clone a recognizable actor, commercial narrator, or living person's voice without documented authorization.

### Speechactors AppSumo lifetime deal

Observed offer:

- approximately $36 one-time;
- 1 million lifetime characters;
- commercial-use audio;
- 300+ voices and 129 languages or accents.

Recommendation:

- possible inexpensive backup or placeholder-voice source;
- audition before purchase;
- do not assume voice quality, emotional control, long-term availability, or API access matches ElevenLabs;
- treat the finite character pool as prepaid usage rather than unlimited lifetime generation.

### Local audio tools

#### REAPER

- $60 discounted license when the user qualifies under Cockos's current revenue/use rules;
- full DAW suitable for dialogue editing, mixing, batch rendering, and SFX organization;
- free updates through the licensed major-version range;
- 60-day full evaluation.

Recommended when the first real audio package begins.

#### Audacity

- free and open source;
- suitable for basic cleanup, trimming, conversion, and quick waveform work.

Use before purchasing a DAW if needs remain simple.

## 3. Image, UI, and asset production

### Affinity

The current unified Affinity application is free.

Recommended uses:

- UI mockups;
- vector and raster editing;
- layout and asset sheets;
- cleanup of AI-generated concepts;
- meaningful human modification before production review.

This should be installed before buying another general-purpose design subscription.

### Aseprite

- approximately $19.99 one-time;
- includes Windows, macOS, Ubuntu, and a Steam key;
- supports sprite sheets, animation timelines, onion skinning, and command-line export.

Recommendation:

- buy only if the project adopts pixel-art or sprite-sheet work that Affinity/Krita cannot handle as efficiently;
- not essential for the current storybook/cel-shaded direction.

### Eagle asset manager

- one-time purchase;
- lifetime updates according to the vendor;
- two-device license;
- Windows and macOS;
- browser extension and local library organization.

Recommendation:

- strong candidate when visual references and generated assets become numerous;
- useful for tags, versions, prompt references, mood boards, and duplicate management;
- repository provenance records remain authoritative even if Eagle is used.

### Blender

- free and open source;
- suitable for 3D blocking, 2.5D dioramas, camera studies, lighting references, render passes, and asset conversion.

### Krita

- free and open source;
- suitable for paint-over, texture, concept cleanup, and frame-by-frame 2D work.

### DaVinci Resolve

- free edition available;
- suitable for trailers, animatics, color work, audio/video assembly, and review exports.

## 4. Lifetime-deal watchlist

Lifetime AI deals carry vendor-survival, model-access, licensing, and credit-policy risk. Purchase only for a defined work package.

### VectorGrove — recommend review

Observed offer:

- approximately $79 one-time;
- 2 million+ vectors and design assets;
- standard and extended licenses;
- 30 downloads per day;
- one user;
- stated commercial permissions.

Potential project use:

- placeholder icons;
- generic UI symbols;
- presentation graphics;
- reference material.

Risks:

- style inconsistency;
- non-refundable offer;
- asset-license details must be saved per selected file;
- do not use assets that imitate protected characters or branded artwork.

### Bulk Image Studio Pro — optional

Observed offer:

- approximately $29.99 one-time;
- Windows desktop batch image workflow;
- uses the purchaser's OpenAI API key;
- reference-image support and project folders.

Recommendation:

- skip for now because ChatGPT and Gemini already cover image generation;
- consider only if high-volume batch prompting becomes a bottleneck;
- API costs are separate from the lifetime software purchase.

### Seelab — watch, do not buy yet

Observed lifetime tiers begin around $79 and include monthly image credits, with custom models appearing on higher tiers.

Recommendation:

- overlaps heavily with ChatGPT and Gemini;
- custom-model consistency could become useful later;
- require a controlled visual-consistency test before purchase.

### Phygital+ — watch, do not buy yet

Observed lifetime tiers begin around $59 and aggregate image, video, and 3D tools.

Recommendation:

- broad tool access is attractive but overlaps current subscriptions;
- model availability and monthly-credit sustainability must be verified;
- do not buy without a missing capability and a test plan.

### Mootion — skip for now

Observed lifetime tiers are several hundred dollars and include video credits and third-party model access.

Recommendation:

- too expensive and duplicative while Google Flow is available;
- revisit only for a specific approved trailer or animatic requirement.

### Picbolt — low priority

Observed offer:

- approximately $49 one-time;
- screenshot mockups and 4K export;
- AI features use a separate API key.

Potential use:

- marketing screenshots and presentation polish.

Not needed for game-content production now.

## 5. Recommended acquisition order

1. Spend nothing additional while ChatGPT Plus and Google AI Plus are underused.
2. Install free Affinity, Blender, Krita, Audacity, and DaVinci Resolve as needed.
3. Use the ElevenLabs Free tier for voice and SFX experiments.
4. Purchase one ElevenLabs Starter month when the Underteller audition pack is ready.
5. Purchase REAPER only when sustained editing and mixing begins.
6. Consider Eagle when the asset library becomes difficult to manage manually.
7. Review VectorGrove only when a concrete icon/vector need exists.
8. Avoid large lifetime AI purchases until an existing tool fails a defined requirement.

## 6. What ChatGPT can directly utilize

ChatGPT can directly:

- create prompts, shot lists, dialogue batches, filenames, manifests, and metadata;
- create and edit images through available image tools;
- generate importable JSON, CSV, Markdown, SVG source, scripts, and code;
- store all briefs and approvals in GitHub;
- inspect outputs uploaded from Gemini, Flow, ElevenLabs, Affinity, Blender, or other tools;
- create revision instructions and production-ready derivative specifications.

ChatGPT cannot directly control an arbitrary installed desktop application in this session.

The practical handoff is:

1. GitHub stores the source brief.
2. The project owner runs the desktop or web tool.
3. The output is uploaded.
4. ChatGPT reviews or transforms it.
5. GitHub stores the approved asset and provenance record.

## 7. Procurement rule

Before purchasing any new service or lifetime deal, record:

- exact use case;
- current price and tier;
- commercial-use terms;
- monthly or lifetime credits;
- export formats;
- watermark policy;
- API or automation availability;
- privacy and training settings;
- refund period;
- vendor continuity risk;
- replacement path if the service closes.
