# Milestone Document Exports

Markdown under `docs/gdd/` and `docs/technical/` is the canonical, diffable source.

Polished DOCX snapshots are produced at meaningful design milestones. The v0.2 DOCX exports are included in the downloadable repository starter package. They are temporarily omitted from the connector-created foundation commit so the repository can be established tonight without Git LFS or a large binary API transfer.

After Git LFS is installed and the asset policy is tested, either:

1. track milestone DOCX exports in LFS, or
2. attach them to tagged GitHub Releases while retaining their source Markdown in Git.

Record the final choice in an ADR before v0.0.3.
