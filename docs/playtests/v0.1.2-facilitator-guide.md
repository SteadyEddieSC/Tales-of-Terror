# v0.1.2 Internal Bundle Facilitator Guide

This guide prepares a session; it does not claim that physical or household validation occurred.

## Bundle preflight

- Extract the complete versioned archive to a normal user-writable folder.
- Windows: use `launch.cmd` or the adjacent `.exe`. Linux: use `./launch.sh` or the adjacent `.x86_64` binary.
- Do not run as administrator. The helper performs no download, installation, update, telemetry, or environment change.
- Open `build_manifest.json` to record the release, source commit, target, and deterministic content digest.
- Confirm X/H opens Help and the Build & Support page shows the expected short source ID, platform, architecture, and `INTERNAL PLAYTEST` classification.
- Connect intended controllers before play. Keyboard remains a development fallback; phones are optional.

## Session route

Observe the existing title -> stable-seat lobby -> confirmation -> briefing -> active tale -> terminal -> ending -> local report export -> rematch route without coaching first. The native Godot host remains authoritative and the complete route must work without phones.

## Report export and exit

At ending, press X/H, move to Playtest Report, then A/Enter. JSON and Markdown are written only to the local Godot user-data folder:

- Windows: `%APPDATA%\Godot\app_userdata\Terror Turn\playtest_exports`
- Linux: `$XDG_DATA_HOME/godot/app_userdata/Terror Turn/playtest_exports`
- Linux default when `XDG_DATA_HOME` is unset: `~/.local/share/godot/app_userdata/Terror Turn/playtest_exports`

`Terror Turn` is the exact current Godot project-folder name and remains a provisional working title. Inspect reports before sharing. Close the native window normally after the session; no background service is installed or retained.

## Evidence discipline

Automation, CI, synthetic controllers, browser E2E, exported-build smoke, and virtual screenshots are not physical evidence. Leave `MANUAL_VALIDATION_RECORD.json` entries as `not_tested` until a human explicitly performs and records the corresponding observation.
