# ADR-0002-logical-resolution — Use a 960×540 logical viewport

- **Status:** Accepted for foundation
- **Date:** July 11, 2026

## Context

The project needs a stable, explainable foundation before gameplay scope expands.

## Decision

The approved detailed 2D/2.5D art and eight-player HUD need more room than 640×360. 960×540 scales exactly to 1080p and 4K.

## Consequences

This decision should be revisited only with measured evidence and a replacement plan.
