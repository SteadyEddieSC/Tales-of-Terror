# ADR-0009-public-repository-and-license — Public visibility with license decision pending

- **Status:** Accepted for foundation
- **Date:** July 11, 2026

## Context

The owner is comfortable with public access and copying in principle as long as others do not charge money, but the project includes software, documents, and art that may need different terms.

## Decision

Keep the repository public. Do not claim the project is open source. Document the intended noncommercial source-available direction, but retain all rights until a specific license is selected. Do not accept outside contributions until contribution and licensing terms are explicit.

## Consequences

The repository can use public GitHub features without requiring a paid plan. Reuse rights remain limited until a license is added, and a future license should distinguish code from creative assets if appropriate.
