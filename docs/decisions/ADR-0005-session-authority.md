# ADR-0005-session-authority — Use native-host gameplay authority first

- **Status:** Accepted for foundation
- **Date:** July 11, 2026

## Context

The project needs a stable, explainable foundation before gameplay scope expands.

## Decision

Cloudflare coordinates rooms/private companion state. High-frequency gameplay remains in the native Godot host during early releases.

## Consequences

This decision should be revisited only with measured evidence and a replacement plan.
