# ADR-0003-compatibility-renderer — Use the Compatibility renderer baseline

- **Status:** Accepted for foundation
- **Date:** July 11, 2026

## Context

The project needs a stable, explainable foundation before gameplay scope expands.

## Decision

The project preserves a future Godot Web export path. Native builds may enable carefully gated enhancements.

## Consequences

This decision should be revisited only with measured evidence and a replacement plan.
