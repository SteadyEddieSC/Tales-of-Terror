# ADR-0006-no-dead-player-downtime — Defeat changes participation rather than ending it

- **Status:** Accepted for foundation
- **Date:** July 11, 2026

## Context

The project needs a stable, explainable foundation before gameplay scope expands.

## Decision

Scenarios provide hostile or supportive afterlife roles, Changed factions, or replacement characters.

## Consequences

This decision should be revisited only with measured evidence and a replacement plan.
