# ADR-0007-documentation-format — Use Markdown as canonical and DOCX as export

- **Status:** Accepted for foundation
- **Date:** July 11, 2026

## Context

The project needs a stable, explainable foundation before gameplay scope expands.

## Decision

Markdown supports review/diff; polished DOCX snapshots support sharing and structured editing.

## Consequences

This decision should be revisited only with measured evidence and a replacement plan.
