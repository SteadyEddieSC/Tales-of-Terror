**Status:** Release candidate on PR #66
**Baseline:** `53a9db5ba1c665ae6ba42873c8aed6e84806890c`
**Scope:** Preproduction content and standard-library tooling only

## Delivered

- 104 unique governed English dialogue entries across seven catalogs.
- Public, seat-private, faction-private, and controlled-private classifications.
- Spooky, Grim, and Gore & Dread variants with plain fallbacks.
- Display, voice, repetition, placeholder, and implementation metadata.
- Bellmarked and Tidebound private objectives.
- Harbor bargain offers, consequences, refusal, and debt collection.
- Drop-in, reconnect, surrogate, takeover, and private recap language.
- Friends Only, Invite Only, Request to Join, spectator, reservation, removal, and host-transition language.
- Item, hazard, card, afterlife, replacement, and ending narration.
- A 24-selection Underteller voice-audition manifest.
- CSV and Markdown audition-export tooling.
- All-catalog validation with cross-file duplicate-key and private-placeholder checks.
- Regression tests and GitHub Actions enforcement.
- Operating-mode, artifact-record, AI-tooling, bundle-procurement, Audio Hero, and original-first visual policies.
