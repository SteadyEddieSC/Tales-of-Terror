# v0.0.2 Input & Display Lab

## Scope

This milestone adds the first interactive controller and display engineering lab. It supports eight independent seats, semantic controller input, stable disconnect reservations, keyboard development fallback, connected-device diagnostics, protected reset, and an adjustable safe-area overlay.

## Controls

| Purpose | Controller | Keyboard fallback |
| --- | --- | --- |
| Join / confirm | A | J / Space / Enter |
| Cancel | B | Escape |
| Navigate | D-pad | WASD / arrows |
| Pause/options | Menu | P |
| Diagnostic test | X | T |
| Safe area | LB / RB | - / + |
| Reset all seats | Hold Y | Hold R |

## Display targets

The project retains a 960×540 logical viewport with `canvas_items` scaling, a kept 16:9 aspect ratio, and the Compatibility renderer. The lab targets 1280×720, 1920×1080, and 3840×2160 output. Essential status text is 15 logical pixels or larger.
