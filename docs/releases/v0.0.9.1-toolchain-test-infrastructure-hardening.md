# v0.0.9.1 Toolchain & Test Infrastructure Hardening

## Delivered

- Advanced Windows development and Linux GitHub Actions validation from Godot 4.7 stable to official checksum-pinned Godot 4.7.1-stable.
- Preserved the Compatibility renderer, 960×540 logical viewport, Godot 4.7 feature-family marker, typed GDScript, Windows/Linux priority, native gameplay authority, and every existing status-check identity.
- Vendored exact GUT 9.7.1 source and licenses, added a deliberately small typed framework/protocol/atomicity/privacy suite, retained every legacy direct test and simulation, emitted JUnit XML, and uploaded it even when the GUT step fails.
- Pinned GDScript Toolkit 4.5.0 and CI Python 3.11.9, excluded only generated/cache and vendored GUT files, and published complete first-party lint/format baselines without rewriting source in CI.
- Documented the toolchain decision, local/CI commands, dependency provenance, baseline gate, Windows evidence, and all deferred manual checks.

## Baseline decision

GDScript Toolkit parsed the entire first-party source tree, but the initial repository baseline is too large for a behavior-preserving tooling PR: 1,235 lint findings, of which 1,210 are line-length, and 66 of 67 initial files would be reformatted. No broad suppressions or weaker typed-GDScript rules were added. Both checks are informational here, and v0.1.0 remains blocked until a separately bounded lint/format cleanup is merged.

## Gameplay and authority position

This release changes tools, tests, CI, and documentation only. It adds no cards, roles, scenarios, combat, enemies, objectives, endings, balancing, player-facing redesign, or network authority. Browser companions still submit bounded intents, the room service remains an ephemeral relay, and native Godot remains the sole gameplay authority.

## Intentionally deferred

No `godot-ci`, export templates, exports, deployment, production Cloudflare changes, storefront work, or v0.1.0 issue are included. Physical controllers, physical phones, household Wi-Fi, television-distance review, native-4K-display inspection, router/firewall variants, live Cloudflare deployment, penetration/security certification, balance/fun evaluation, and long-session testing are not claimed.
