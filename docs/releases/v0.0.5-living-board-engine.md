# v0.0.5 Living Board Engine

## Delivered

- A validated authored Lantern House board with five named spaces, multi-area region geometry, eight spawn locations, and five typed/directional connectors.
- Separate authoritative runtime reveal, hazard, feature, blocker, connector, occupancy, revision, rejection, and history state.
- Stable pawn-to-space mapping with deterministic overlap/boundary handling and explicit `outside_board` state.
- Deterministic direct-connectivity, traversability, reachability, shortest-path, blocked-crossing, and required-disconnection queries.
- Atomic reveal, connector, hazard, feature, and blocker mutations with lowest-seat simultaneous conflict arbitration.
- Versioned JSON-compatible in-memory snapshots with atomic malformed/unsupported rejection.
- Signal-driven board-debug rendering and expanded diagnostics using text, symbols, patterns, outlines, and Roman numeral occupancy alongside color.
- Pinned Godot 4.7 CI coverage for the complete Living Board regression suite.
