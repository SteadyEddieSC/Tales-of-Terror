## Security and privacy position

Native Godot is the only gameplay authority. The service is a replaceable ephemeral relay and has no rules, player profile, campaign history, room enumeration, analytics, advertising, account, chat, voice, camera, biometric, or fingerprinting system. Platform TLS is required outside labeled localhost development. The prototype is not end-to-end encrypted against the relay operator and is not a production security certification.
