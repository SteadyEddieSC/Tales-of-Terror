# Visual Baseline

`haunted_board_visual_baseline.webp` is an optimized repository copy of the approved visual-direction baseline for board readability, shared-screen composition, mood, player identification, and 2D/2.5D presentation. The higher-resolution source remains preserved in the milestone starter package and DOCX exports.

It is not a final production asset. Final art must be original and should sit between the concept’s colorful readability and a weightier hand-inked storybook-horror presentation, without copying Darkest Dungeon characters, UI, compositions, or exact rendering techniques.
