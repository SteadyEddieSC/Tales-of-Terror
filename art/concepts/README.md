# Visual Baseline

The approved haunted-board concept is preserved in the downloadable v0.2 starter package and polished DOCX exports. The connector-created foundation commit intentionally omits the binary image until the project adopts either Git LFS or a tagged Release-asset policy.

The concept establishes board readability, shared-screen composition, mood, player identification, and 2D/2.5D presentation. It is not a final production asset. Final art must be original and should sit between the concept’s colorful readability and a weightier hand-inked storybook-horror presentation, without copying Darkest Dungeon characters, UI, compositions, or exact rendering techniques.
